# pizza-sales-sql-analysis
SQL-based data analysis project on pizza sales dataset to extract business insights using MySQ

## 📌 Project Overview
This project focuses on analyzing a real-world pizza sales dataset using SQL to extract meaningful business insights. The analysis helps understand sales performance, customer preferences, and revenue trends.

## 🎯 Objectives
- Analyze total number of orders
- Calculate total revenue generated
- Identify top-selling pizzas
- Understand customer preferences (size, category)
- Analyze sales trends over time

## 📊 Dataset Description
The dataset consists of the following tables:
- **Orders** – Contains order date and time
- **Order Details** – Contains quantity and pizza IDs
- **Pizzas** – Contains price and size information
- **Pizza Types** – Contains pizza names and categories

## 🔍 Key Analysis Performed
- Total number of orders placed
- Total revenue generated from pizza sales
- Highest-priced pizza
- Most common pizza size ordered
- Top 5 most ordered pizza types
- Category-wise quantity distribution
- Hour-wise order analysis (peak hours)
- Average pizzas ordered per day
- Top pizzas based on revenue
- Revenue contribution by category
- Cumulative revenue over time
- Top pizzas per category (advanced SQL)

## 🛠️ Tools & Technologies Used
- SQL (MySQL)
- Data Analysis

## 💡 Key Insights
- Peak ordering hours identified during lunch and evening time
- Certain pizza categories contribute more to total revenue
- Large size pizzas are the most preferred by customers
- Few pizza types dominate overall sales
